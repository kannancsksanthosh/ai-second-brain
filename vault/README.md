# AI Second Brain

## Overview

This repository contains my personal AI Second Brain.

The purpose of this project is to build a structured knowledge system that stores my work, projects, learning, goals, and personal notes in Markdown format. Instead of relying on AI chat history, all important information is stored as files that can be searched, updated, and understood over time.

This is the **Basic Version (Phase 1)** of the project.

---

## Objectives

* Build a long-term personal knowledge base.
* Organize information using Markdown.
* Make information easy for both humans and AI to understand.
* Keep knowledge updated instead of rewriting it.
* Create a strong foundation for the future AI Second Brain Pro.

---

---

## Vault Purpose

The `vault/` directory is the heart of this project.

Every important piece of knowledge should live inside the vault.

Examples include:

* Career information
* Project documentation
* Learning notes
* Goals and planning
* Personal knowledge
* Useful resources

---

## Project Rules

1. One topic per Markdown file.
2. Do not duplicate information across files.
3. Update existing notes instead of creating duplicates.
4. Use clear and descriptive file names.
5. Link related notes using Markdown or wiki links when appropriate.
6. Keep notes simple, structured, and easy to understand.
7. The `INDEX.md` file should always reflect the current structure of the vault.
